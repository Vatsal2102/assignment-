getWeatherCondition(array<string|int, mixed> $formattedWeather) : string

This function analyzes the weather data to return a standardized weather condition string used for styling and icon selection. It considers:

  • Main weather type (clear, clouds, rain, etc.)
  • Time of day (day or night for clear conditions)
Parameters
$formattedWeather : array<string|int, mixed>

Associative array containing current weather information

Return values
string —

Standardized weather condition (clear-day, clear-night, clouds, rain, snow)
